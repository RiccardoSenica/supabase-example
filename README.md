# Supabase configuration

Initialize Supabase

```bash
supabase init
```

Login to Supabase

```bash
supabase login
```

Link to your project

```bash
supabase link --project-ref YOUR-PROJECT-REF
```

Pull remote changes

```bash
supabase db pull
```

Create a new migration

```bash
supabase migration new YOUR-MIGRATION-NAME
```

Apply migrations

```bash
supabase db push
```

Reset database

```bash
supabase db reset
```
